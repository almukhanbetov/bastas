# BAS TAS — modern redesign

## Открыть локально

```bash
cd bastas-modern
python3 -m http.server 8000
```

Откройте `http://localhost:8000`.

## Важно
Фотографии подключены с Unsplash по URL. Для полной автономности скачайте свои реальные фото камня/работ и замените URL в HTML-файлах на локальные изображения из `assets/images/`.
